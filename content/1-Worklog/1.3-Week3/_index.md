**Week 3:** 4–10 May 2026

### Week 3 Objectives:

- Reinforce EC2 deployment on Windows/Linux and understand core virtual server components.
- Deploy applications connected to RDS, perform backup/restore, and build scalable systems with Launch Template, Load Balancer, and Auto Scaling Group.
- Monitor and control AWS spending using AWS Budgets (Cost, Usage, RI, Savings Plans).

- Know how to deploy applications on EC2, connect to and manage RDS databases, and perform backup and restore. Understand how to use Load Balancer and Auto Scaling to build scalable, highly available systems, and how to use AWS Budgets to monitor and control AWS service costs.
